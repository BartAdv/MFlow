This is the code of the wiki widget,  that render this page:

wiki =  do
  pag <-  getRestParam `onNothing` return "index"
  page $
   (docTypeHtml $  El.head $ El.title << pag)
        ++> ( h1 ! At.style "text-align:center" 
             <<<  tFieldEd "editor" (wikip ++ pag ++ "title.html") (fromString pag))
        **> tFieldEd "editor" (wikip ++ pag ++ "body.html") "Enter the body"


wikip="wiki/"

getRestParam read the next segment of the REST path. If no remaining segment, "index" is used.

tfieldEd is HTML text that can be edited by the "editor" user. It present the static page defined in the second parameter.

The page use blaze-html combinators and MFlow combinators.

There are two fields, the title of the page and the body of the page.  This content is in "wiki/indexbody.html"
